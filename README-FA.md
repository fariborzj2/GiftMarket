# سیستم عمده و جزئی فروش گیفت کارت

یک پلتفرم چندزبانه برای فروش گیفت کارت، مناسب هر دو حالت عمده و جزئی، ساخته شده با بک‌اند سبک PHP و معماری پلاگینی برای توسعه و نگهداری بلندمدت.

## امکانات اصلی
- مدیریت فروش عمده و جزئی گیفت کارت
- سیستم مدیریت مشتریان (پروفایل‌ها، تاریخچه سفارشات، وضعیت‌ها)
- مدیریت سفارشات و تراکنش‌ها
- معماری پلاگینی برای توسعه ماژولار
- بک‌اند سبک و بهینه‌شده PHP
- پشتیبانی چندزبانه (آماده i18n)
- تم روشن و تاریک
- رابط کاربری کاملاً رسپانسیو
- بهینه‌شده برای SEO
- فرانت‌اند با موتور قالب `.tpl`
- نصب آسان با نصب‌کننده (installer) مرحله‌ای

## پنل مدیریت
- پنل حرفه‌ای PHP برای مدیریت سایت
- استفاده از موتور قالب `.tpl` برای نمایش پنل
- کنترل دسترسی مبتنی بر نقش (مدیر / کارمند)
- مدیریت محصولات، دسته‌بندی‌ها، قیمت‌ها و موجودی
- مدیریت و دسته‌بندی مشتریان
- مدیریت و پیگیری سفارشات
- مدیریت ماژول‌ها و پلاگین‌ها
- تنظیمات سایت و مدیریت محتوا

## پرداخت و توسعه‌پذیری
- در حال حاضر درگاه پرداخت آنلاین یکپارچه ندارد
- معماری پلاگینی امکان اضافه کردن آسان درگاه‌های پرداخت در آینده را فراهم می‌کند
- توسعه‌دهندگان می‌توانند پلاگین پرداخت سفارشی ایجاد کنند بدون اینکه هسته سیستم را تغییر دهند

## معماری سیستم
- بک‌اند سبک PHP با حداقل وابستگی‌ها
- سیستم پلاگین با استفاده از Hooks و Events برای توسعه امن
- جداسازی منطق هسته از پلاگین‌ها برای جلوگیری از تغییر هسته
- ساختار آماده برای API و REST/JSON در آینده
- مدیریت مرکزی تنظیمات

## نصب
سیستم شامل نصب‌کننده‌ای مرحله‌ای است که کاربر را در طی فرآیند راه‌اندازی، شامل پیکربندی دیتابیس و تنظیمات اولیه، راهنمایی می‌کند.

نصب از طریق رابط وب مرحله‌ای انجام می‌شود و سایت در چند قدم ساده آماده استفاده خواهد بود.

## موارد استفاده
- توزیع‌کنندگان عمده گیفت کارت
- فروشگاه‌های خرده‌فروشی گیفت کارت
- پلتفرم‌های محصولات دیجیتال چندزبانه
- بازارهای سفارشی گیفت کارت

## اهداف پروژه
- حفظ بک‌اند سبک و قابل نگهداری
- امکان توسعه آسان از طریق پلاگین‌ها، از جمله درگاه‌های پرداخت آینده
- جلوگیری از وابستگی به فریم‌ورک خاص
- ایجاد پایه‌ای تمیز و مقیاس‌پذیر برای استفاده واقعی
